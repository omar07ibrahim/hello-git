Git Lab Notes - Exercise 1

Key Concepts:
- Git is a distributed version control system
- Every clone has the full history of the project
- Commits create snapshots of the project at specific points in time

Initial Thoughts:
Learning Git helps me collaborate with other developers and track changes in my code over time.
Version control is essential for modern software development.

## Summary
Version control with Git enables better collaboration and code management across teams.
Mastering Git commands is a fundamental skill for any developer.

## Insights
- Branching allows parallel development without affecting the main codebase
- Commits should be atomic and have meaningful messages
- Remote repositories enable team collaboration
